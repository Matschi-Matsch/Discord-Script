This repository contains the **main script for a modular Discord bot**.
The project was originally created for personal use and was later made **public to help other developers**.

# Installation

1. Download  this repository:

2. Make sure your server has the following structure:

```
/home/container/
│
├── main.py
└── Modules/
```

If you are using **BisectHosting**, follow these steps:

* Upload the file **main.py** to
  `/home/container/`
* Move the **Modules** folder into the same directory.

Your bot should now be ready to start.

# Security Notice

⚠️ **Never leak your Discord bot token.**

* Never share your **bot token publicly**.
* Do **not upload code containing your token to GitHub**.
* Store tokens in **environment variables or private config files**.
* If your token gets leaked, **reset it immediately in the Discord Developer Portal**.

A leaked token can allow others to **take full control of your bot**.

More information:
https://discord.com/developers/docs/topics/security
